# 100 Days of Data Structures & Algorithms (DSA) 🚀

A dedicated challenge to sharpen my problem-solving skills, optimize code efficiency, and master fundamental computer science concepts using **TypeScript** and **Vitest**.

## 🛠️ Tech Stack

- **Language:** TypeScript
- **Runtime:** Node.js
- **Testing:** Vitest
- **Environment:** ESM (EcmaScript Modules)

## 🚀 How to Run Locally

### 1.Clone the repository:\*\*

```bash
git clone https://github.com/Mikkybeardless/dsa-challenge.git
cd dsa-challenges
```

### 2. Install dependencies:

```bash
npm install
```

### 3. Run tests for all challenges:

```bash
npm test
```

### 4. Run tests in watch mode (for development):

```bash
 npm run dev
```
